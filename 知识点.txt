一。git的分支
	1.分支的基本操作
		1.查看分支 git branch 
			*表示当前处于哪个分支

		2.创建分支 git branch 分支名称  
			创建出来的分支 与当前分支内容一样 当前分支中的提交信息 在新创建的分支中都能看到

		3.切换分支 git checkout 分支名称 
			创建并切换分支 git checkout -b 分支名称

		4.合并分支 git merge 分支名称

		5.删除分支 git branch -d 分支名称

	分支冲突：
		在两个分支中 都修改了同一行的内容 并且提交 此时再合并分支就会产生冲突 这个时候就需要手动解决 ：决定需要保留那个分支的内容
		解决方式：
			1.决定保留哪些内容 手动
			2.git add -A
			3.git commit -m ""


二。github
	将本地代码推送到github的远程仓库
		git push 远程地址 分支名称

	从远程仓库拉取代码
		git pull 远程地址 分支名称
